Arch Linux on WSL

要经常在 Linux 和 -辣鸡- Windows 之间来回折腾真的是很难受，而且有时候还只是要在 Shell 上跑点脚本或者命令。

所幸有 WSL 这个好东西，但是 Micro soft Store 能下载的只有 Ubuntu。 学习了一下可以用一些特殊的技巧变成 Arch ，再在
AUR 上下东西就方便得多的了。 1. 先在 Store 里下载 Ubuntu，并且在 Powershell 下把默认用户改成 root ubuntu —-config user root 2. 从 USTC 或者清华的镜像站下载 archlinux-bootstrap， 为了方便建议放在任意盘符的根目录下。

1. Ubuntu 下 `cd /mnt/c` 并 `tar -zxvf`（不要在 Windows 下解压，否则会导致权限问题）
2. 在 Windows Explorer 中将 Ubuntu 目录下的 LocalState/rootfs/中的以下文件删除。并且用 archlinux-bootstrap

中的同名文件和目录替换。

* bin
* etc
* lib
* lib64
* sbin
* usr
* var 3. 从运行 Arch 的电脑或者软件源下载 fakeroot-tcp 并且放在文件系统中一个方便的目录下 （如果你想使用

AUR ，这一步是必要的） 4. 再次进入 Ubuntu 并运行 pacman-key —init pacman-key —populate archlinux pacman -U path-to-fakeroot-tcp pacman -Syyu base base-devel 5. 新建一个用户并且添加到 sudoers
